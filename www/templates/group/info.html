主面板 / {{ source.name }} / {{ group.name }}

Request URL

http://stat.berwin.me/api/v1/content

Method

POST

Parameters

{
    groupID : '{{group._id}}',
    sourceID : '{{source._id}}',
    token: '{{source.token}}'
    data : {
        {{item.key}}: 'test',

        value: '{{group.values[0].key}}'
    }
}

注：所有ID及参数均为真实有效的实际参数

  • groupID : groupID
  • sourceID: sourceID
  • token : 为了验证身份用，token在主面板信息中查看。
  • data：统计的关键数据
    • ?: 这是一个 键(key) : 值(value)，键(key)为创建group(蓝块) 时，所填写的key，值(value)根据需求填写 （这里不明白没关系，下面会给出例子）
    • ?: ... 同上（根据创建group(蓝块) 时配置的key的数量，这里可能有N多个）
    • value: 值为创建group(蓝块) 时配置的value中选一个

更详细的API使用说明可以去帮助中心查看。
