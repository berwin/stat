主面板 / {{ source.name }} / {{ group.name }}

Request URL

http://stat.berwin.me/api/v1/content

Method

POST

Parameters

{
    groupID : '{{group._id}}',
    sourceID : '{{source._id}}',
    token: '{{source.token}}'
    data : {
        {{item.key}}: 'test',

        value: '{{group.values[0].key}}'
    }
}
  • groupID : groupID
  • sourceID: sourceID
  • token : 为了验证身份用，token在主面板信息中查看。
  • data : 统计的关键数据
    • code: key 为动态key 为蓝色块里面设置的key，value 根据需求填写 ... 这里可能有N多个Key (如果有多个Key必须把所有key都写在参数里)
    • value: value 为固定key(随意填写将无法统计) 值根据蓝色块里面设置的value列表中选一个
